# Password Strength Analyzer

## Project Overview
The Password Strength Analyzer is a Python-based application that evaluates the strength of user-generated passwords based on predefined security policies.

## Objective
To analyze password strength using:
- Length validation
- Character complexity checks
- Regular expressions
- Score-based evaluation

## Features
- Checks minimum length (8 characters)
- Validates uppercase and lowercase letters
- Ensures inclusion of digits
- Requires special characters
- Provides feedback for weak passwords
- Calculates security score percentage

## Technologies Used
- Python 3
- Regular Expressions (re module)
- Command Line Interface
